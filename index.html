Probador APIs

Por ahora el probador de APIs no está disponible en dispositivos móviles.
En el campo de request.body (izquierda), los datos deben ser escritos con formato JSON (con los keys entre comillas dobles y sin comas finales). Se pueden encontrar ejemplos de usos en README.md -> Github
Para poder utilizar el probador de API con usuario creado recientemente, su usuario y contraseña deberán ser la misma.
